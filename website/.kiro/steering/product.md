# Product Overview

## Crypto Launchpad MCP

A minimalist, Apple-inspired landing page for the Crypto Launchpad MCP tool - an AI-powered token deployment and liquidity management system built on the Model Context Protocol (MCP).

### Key Features

- AI-powered token deployment using natural language commands
- Secure wallet integration with EIP-6963 wallet discovery
- Multi-chain support (Ethereum, Solana)
- Complete Uniswap DEX functionality
- Smart contract templates from OpenZeppelin
- MCP protocol integration for AI assistants

### Target Audience

- Crypto developers and traders
- AI assistant users (Claude, Cursor, VS Code)
- DeFi protocol builders
- Token creators and liquidity providers

### Design Philosophy

- Apple-like minimalist aesthetic
- Clean typography with generous whitespace
- Smooth animations and micro-interactions
- Performance-first approach
- Accessibility compliance (WCAG AA)
